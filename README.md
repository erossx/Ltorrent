# 🌟 Advanced Torrent Client

PySide6와 libtorrent를 사용한 고급 토렌트 클라이언트입니다. 보안과 익명성을 중시하는 사용자를 위한 다양한 기능을 제공합니다.

## 📋 주요 기능

### 기본 토렌트 기능
- 토렌트 파일 및 마그넷 링크 지원
- 실시간 다운로드/업로드 속도 표시
- 진행률 표시 및 토렌트 관리
- 일시정지/재개/제거 기능
- 탭 기반 다크 테마 UI

### ⚡ 고급 제어 기능
- **속도 제한**: 업로드/다운로드 속도 제한 (KB/s 단위)
- **자동 종료**: 모든 다운로드 완료 시 컴퓨터 자동 종료
- **실시간 통계**: 전체 업로드/다운로드 통계

### 🔒 보안 강화 기능
- **피어 간 통신 암호화**: 데이터 전송 암호화
- **랜덤 포트**: 자동 랜덤 포트 선택 (49152-65535)
- **IP 필터링**: 악성 IP 범위 자동 차단 + 수동 IP 차단
- **DHT 제어**: 익명성 향상을 위한 DHT 비활성화 옵션
- **보안 로그**: 모든 보안 이벤트 실시간 기록 및 표시
- **파일 해시 검증**: SHA256을 통한 파일 무결성 확인

### 🧅 익명성 & 프록시 지원
- **익명 모드**: DHT/LSD 비활성화, User-Agent 변경
- **프록시 지원**: HTTP, SOCKS4/SOCKS5, 인증 프록시
- **Tor 자동 연결**: 한 클릭으로 Tor 네트워크 연결
- **실시간 상태 모니터링**: Tor 연결 상태 실시간 표시

## 🛠 설치

### 1. 가상환경 생성 (권장)

```bash
python3 -m venv torrent_env
source torrent_env/bin/activate  # macOS/Linux
# 또는
torrent_env\Scripts\activate     # Windows
```

### 2. 의존성 설치

```bash
pip install -r requirements.txt
```

### 3. Tor 설치 (익명성 기능 사용 시)

**macOS (Homebrew)**:
```bash
brew install tor
brew services start tor
```

**Ubuntu/Debian**:
```bash
sudo apt-get install tor
sudo systemctl start tor
```

**Windows**:
[Tor Browser](https://www.torproject.org/download/) 설치 후 전문가 번들 사용

## 🚀 실행

```bash
# 가상환경 활성화
source torrent_env/bin/activate

# 프로그램 실행
python3 main.py
```

## 📖 사용법

### 기본 토렌트 관리

#### 토렌트 파일 추가
1. "토렌트 파일 추가" 버튼 클릭 또는 `Ctrl+O`
2. `.torrent` 파일 선택
3. 다운로드 폴더 선택

#### 마그넷 링크 추가
1. "마그넷 링크 추가" 버튼 클릭 또는 `Ctrl+M`
2. 마그넷 링크 입력
3. 다운로드 폴더 선택

#### 토렌트 제어
- **일시정지**: 토렌트 선택 → "일시정지" 버튼
- **재개**: 일시정지된 토렌트 선택 → "재개" 버튼  
- **제거**: 토렌트 선택 → "제거" 버튼 (파일 삭제 여부 선택 가능)

### 고급 기능 설정

#### 속도 제한
1. "통계 & 설정" 탭으로 이동
2. "업로드 속도 제한" 또는 "다운로드 속도 제한" 입력 (KB/s)
3. "속도 제한 적용" 버튼 클릭

#### 자동 종료
1. "통계 & 설정" 탭에서 "모든 다운로드 완료 시 컴퓨터 종료" 체크박스 활성화
2. 모든 토렌트가 완료되면 5초 후 자동 종료

### 보안 설정

#### IP 필터링
1. "보안" 탭으로 이동
2. "IP 필터 활성화" 체크박스 활성화
3. 수동 IP 차단: IP 주소 입력 → "IP 차단" 버튼

#### 익명성 강화
1. "보안" 탭에서 "DHT 비활성화" 체크박스 활성화
2. "익명 모드 활성화" 체크박스 활성화

### Tor 연결

#### 자동 연결
1. "보안" 탭으로 이동
2. "🧅 Tor 자동 연결" 버튼 클릭
3. Tor 상태가 "✅ 연결됨"으로 변경되는지 확인

#### 수동 프록시 설정
1. 프록시 타입 선택 (HTTP/SOCKS4/SOCKS5)
2. 서버 주소와 포트 입력 (예: 127.0.0.1:9050)
3. 필요 시 사용자명/비밀번호 입력
4. "프록시 적용" 버튼 클릭

## ⌨️ 단축키

- `Ctrl+M`: 마그넷 링크 추가
- `Ctrl+Q`: 프로그램 종료

## ⚠️ 주의사항

### 법적 책임
- 이 소프트웨어는 교육 및 정당한 용도로만 사용하세요
- 저작권이 있는 콘텐츠의 불법 다운로드는 금지됩니다
- 사용자의 행동에 대한 모든 법적 책임은 사용자에게 있습니다

### 보안 고려사항
- Tor 사용 시에도 완전한 익명성은 보장되지 않습니다
- VPN과 함께 사용하면 보안이 더욱 강화됩니다
- 의심스러운 토렌트 파일은 다운로드하지 마세요

### 네트워크 설정
- 방화벽에서 포트 허용이 필요할 수 있습니다
- 토렌트는 네트워크 대역폭을 많이 사용합니다
- ISP의 토렌트 제한 정책을 확인하세요

## 🔧 문제 해결

### libtorrent 설치 오류
```bash
# macOS
brew install libtorrent-rasterbar
pip install python-libtorrent

# Ubuntu/Debian
sudo apt-get install python3-libtorrent

# Windows
pip install python-libtorrent
```

### Tor 연결 오류
```bash
# Tor 서비스 상태 확인
brew services list | grep tor  # macOS
systemctl status tor           # Linux

# Tor 재시작
brew services restart tor      # macOS
sudo systemctl restart tor     # Linux
```

### 권한 오류
- 다운로드 폴더에 쓰기 권한이 있는지 확인
- macOS에서는 "보안 및 개인 정보 보호" 설정 확인

### 포트 충돌 오류
- 다른 토렌트 클라이언트가 실행 중인지 확인
- 프로그램이 자동으로 다른 포트를 선택하도록 대기

## 🛡️ 보안 팁

1. **항상 VPN 사용**: Tor와 VPN을 함께 사용하여 이중 보안
2. **정기적인 IP 확인**: 실제 IP가 노출되지 않는지 확인
3. **안전한 토렌트만 다운로드**: 출처가 확실한 토렌트만 사용
4. **보안 로그 모니터링**: 의심스러운 활동이 있는지 정기 확인
